# Student Portfolio Example

แฟ้มสะสมผลงานออนไลน์ตัวอย่างของนักเรียนระดับมัธยมศึกษาตอนปลาย  
จัดทำเพื่อใช้เป็นตัวอย่างการสร้าง Portfolio บน GitHub สำหรับการเรียนรู้เรื่อง TCAS Portfolio, Digital Portfolio และ Future Skills

> ข้อมูลทั้งหมดใน repository นี้เป็นข้อมูลสมมุติ ใช้เพื่อการเรียนรู้เท่านั้น

## ข้อมูลเจ้าของ Portfolio

| รายการ | ข้อมูล |
|---|---|
| ชื่อ | นภัสสร ใจดี |
| ชื่อเล่น | พลอย |
| ระดับชั้น | ม.5 |
| โรงเรียน | โรงเรียนตัวอย่าง |
| ความสนใจ | จิตวิทยา การสื่อสาร งานอาสา การทำสื่อสร้างสรรค์ |
| คณะที่สนใจ | จิตวิทยา / นิเทศศาสตร์ / ครุศาสตร์ |

## ทำไมใช้ GitHub เป็น Portfolio

GitHub ไม่ได้มีไว้สำหรับนักเขียนโปรแกรมเท่านั้น แต่ยังใช้เป็นแฟ้มสะสมผลงานออนไลน์ได้ เพราะสามารถจัดเก็บผลงานเป็นหมวดหมู่ เขียนบันทึกการเรียนรู้ แสดงหลักฐาน และอัปเดตพัฒนาการได้ต่อเนื่อง

Portfolio ตัวอย่างนี้เน้นให้เห็นว่า นักเรียนทุกสายสามารถเล่าเรื่องของตัวเองได้ผ่าน

- ประวัติและความสนใจ
- เป้าหมายการเรียนต่อ
- โครงงานหรือผลงานที่เคยทำ
- กิจกรรมและบทบาทที่รับผิดชอบ
- เกียรติบัตรหรือหลักฐานการเรียนรู้
- Reflection หรือการสะท้อนคิดจากประสบการณ์

## วิธีใช้ Portfolio ตัวอย่างนี้

1. อ่านภาพรวมจากไฟล์นี้ก่อน
2. เปิดไฟล์ [ABOUT-ME.md](ABOUT-ME.md) เพื่อดูตัวอย่างการแนะนำตัว
3. ดู [GOAL-MAP.md](GOAL-MAP.md) เพื่อเข้าใจการเชื่อมโยงความสนใจกับคณะที่อยากเรียน
4. เปิดโฟลเดอร์ [projects](projects/) เพื่อดูตัวอย่างการเขียนผลงาน
5. เปิด [TCAS-SUMMARY.md](TCAS-SUMMARY.md) เพื่อดูตัวอย่างการสรุปประเด็นสำหรับยื่นมหาวิทยาลัย

## หลักการเขียน Portfolio

- เขียนให้จริงใจและชัดเจน
- ไม่จำเป็นต้องมีผลงานใหญ่ แต่ควรอธิบายสิ่งที่เรียนรู้ได้
- แสดงบทบาทของตัวเองในกิจกรรมนั้น ๆ
- เชื่อมโยงผลงานกับคณะหรือสาขาที่สนใจ
- มี Reflection ว่าได้เรียนรู้อะไรและจะพัฒนาต่ออย่างไร

## หมายเหตุสำหรับนักเรียน

นักเรียนสามารถคัดลอกโครงสร้างนี้ไปปรับใช้ได้ แต่ควรเปลี่ยนเนื้อหาให้เป็นประสบการณ์จริงของตนเอง ไม่ควรคัดลอกข้อมูลสมมุติไปใช้ยื่นจริง
