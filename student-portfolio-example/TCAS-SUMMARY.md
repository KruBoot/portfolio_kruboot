# TCAS Portfolio Summary

ไฟล์นี้เป็นตัวอย่างการสรุปข้อมูลสำคัญสำหรับใช้เตรียมยื่น TCAS รอบ Portfolio

> ข้อมูลทั้งหมดเป็นตัวอย่างสมมุติ นักเรียนควรปรับให้ตรงกับประสบการณ์จริงและเกณฑ์ของมหาวิทยาลัยที่ต้องการยื่น

## ข้อมูลผู้สมัคร

| รายการ | ข้อมูล |
|---|---|
| ชื่อ | นภัสสร ใจดี |
| ชื่อเล่น | พลอย |
| ระดับชั้น | ม.5 |
| โรงเรียน | โรงเรียนตัวอย่าง |
| สายการเรียน | ตัวอย่างกลาง สามารถปรับใช้ได้ทุกสาย |
| คณะที่สนใจ | จิตวิทยา / นิเทศศาสตร์ / ครุศาสตร์ |

## Profile สั้นสำหรับหน้าแรก Portfolio

นักเรียนระดับชั้น ม.5 ที่สนใจจิตวิทยา การสื่อสาร และการทำสื่อสร้างสรรค์ มีประสบการณ์ทำกิจกรรมที่เกี่ยวข้องกับการรับฟังเพื่อน การรณรงค์เรื่องสุขภาพใจ และการเขียนบันทึกการอ่านเพื่อพัฒนาตนเอง

## จุดเด่นของผู้สมัคร

- มีความสนใจชัดเจนด้านการเข้าใจมนุษย์และการสื่อสาร
- มีผลงานที่เชื่อมโยงกับสุขภาพใจวัยรุ่น
- มีประสบการณ์ทำกิจกรรมร่วมกับเพื่อนและครู
- เขียน Reflection เพื่อสะท้อนการเรียนรู้จากประสบการณ์
- ใช้ GitHub จัดระบบแฟ้มสะสมผลงานออนไลน์ได้

## ผลงานเด่น 3 ชิ้น

| ผลงาน | ประเภท | เหตุผลที่เลือก |
|---|---|---|
| Peer Support Campaign | กิจกรรม / งานอาสา | แสดงความสนใจด้านจิตวิทยา การฟัง และการดูแลเพื่อน |
| คลิปสั้นเรื่องสุขภาพใจ | สื่อสร้างสรรค์ | แสดงทักษะการสื่อสารและการผลิตสื่อสำหรับวัยรุ่น |
| Reading Journal | การเรียนรู้ด้วยตนเอง | แสดงความพยายามศึกษาต่อเนื่องและการคิดสะท้อนตนเอง |

## ความเชื่อมโยงกับคณะจิตวิทยา

Portfolio นี้แสดงให้เห็นความสนใจด้านพฤติกรรม ความคิด และอารมณ์ของมนุษย์ ผ่านกิจกรรมรับฟังเพื่อน การเรียนรู้เรื่องสุขภาพใจ และการเขียน Reflection นักเรียนยังเข้าใจขอบเขตของการช่วยเหลือเบื้องต้น และเห็นความสำคัญของการส่งต่อผู้เชี่ยวชาญเมื่อจำเป็น

## ความเชื่อมโยงกับคณะนิเทศศาสตร์

ผลงานคลิปสั้นและสื่อรณรงค์สะท้อนความสามารถในการเลือกประเด็นที่เกี่ยวข้องกับสังคมวัยรุ่น แล้วสื่อสารให้เข้าใจง่าย นักเรียนได้ฝึกเขียนบท เล่าเรื่อง และคิดถึงผลกระทบของสื่อต่อผู้ชม

## ความเชื่อมโยงกับคณะครุศาสตร์ / ศึกษาศาสตร์

กิจกรรมแนะแนวน้องและงานอาสาแสดงให้เห็นความสนใจในการช่วยเหลือผู้อื่น การอธิบายเรื่องยากให้ง่ายขึ้น และการออกแบบกิจกรรมที่เหมาะกับผู้เรียน ซึ่งเป็นพื้นฐานสำคัญของงานด้านการศึกษา

## สิ่งที่ควรเตรียมเพิ่มเติมก่อนยื่นจริง

- ตรวจสอบเกณฑ์รับสมัครของแต่ละมหาวิทยาลัย
- เลือกผลงานให้ตรงกับคณะ ไม่จำเป็นต้องใส่ทุกอย่าง
- เตรียมไฟล์หลักฐาน เช่น รูปภาพ เกียรติบัตร และหนังสือรับรอง
- เขียน Statement of Purpose หรือแนะนำตัวให้สอดคล้องกับ Portfolio
- ให้ครูที่ปรึกษาหรือครูแนะแนวช่วยตรวจเนื้อหา

## ตัวอย่างประโยคสรุปสำหรับใช้สัมภาษณ์

> Portfolio นี้แสดงเส้นทางการค้นหาตัวเองของฉันจากความสนใจเรื่องการฟังและการเข้าใจเพื่อน ไปสู่การทำกิจกรรมและสื่อสร้างสรรค์เกี่ยวกับสุขภาพใจวัยรุ่น ฉันยังต้องพัฒนาต่ออีกมาก แต่ผลงานเหล่านี้ทำให้ฉันมั่นใจว่าอยากเรียนต่อในสาขาที่เกี่ยวข้องกับการเข้าใจมนุษย์ การสื่อสาร และการพัฒนาผู้คน
