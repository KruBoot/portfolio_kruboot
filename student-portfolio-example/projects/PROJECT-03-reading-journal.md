# Project 03: Reading Journal

## ชื่อผลงาน

บันทึกการอ่าน "เข้าใจตัวเอง เข้าใจคนอื่น"

## ประเภทผลงาน

บันทึกการเรียนรู้ / Reading Journal / Self-Learning

## ระยะเวลา

กันยายน - พฤศจิกายน 2569

## แนวคิดของผลงาน

ฉันเริ่มทำ Reading Journal เพื่อฝึกอ่านบทความและหนังสือที่เกี่ยวข้องกับจิตวิทยา การสื่อสาร และการพัฒนาตนเอง จากนั้นสรุปเป็นภาษาของตัวเอง พร้อมเขียนสิ่งที่นำไปใช้ได้จริงในชีวิตประจำวัน

## วัตถุประสงค์

- ฝึกอ่านและจับใจความสำคัญ
- ฝึกแยกข้อเท็จจริง ความคิดเห็น และประสบการณ์ส่วนตัว
- สะสมความรู้ที่เกี่ยวข้องกับคณะที่สนใจ
- พัฒนาทักษะการเขียน Reflection

## ตัวอย่างหัวข้อที่บันทึก

| ครั้ง | หัวข้อที่อ่าน | สิ่งที่ได้เรียนรู้ |
|---|---|---|
| 1 | การฟังอย่างตั้งใจ | การฟังที่ดีต้องให้พื้นที่อีกฝ่าย ไม่รีบแทรกหรือสรุปแทน |
| 2 | ความเครียดของวัยรุ่น | ความเครียดเกิดได้จากหลายปัจจัย และแต่ละคนรับมือไม่เหมือนกัน |
| 3 | การสื่อสารแบบไม่ทำร้ายกัน | การเลือกคำพูดมีผลต่อความรู้สึกและความสัมพันธ์ |
| 4 | การรู้จักตนเอง | การเขียนบันทึกช่วยให้เห็นความคิดและอารมณ์ของตัวเองชัดขึ้น |

## รูปแบบบันทึกที่ใช้

แต่ละบันทึกจะมีหัวข้อดังนี้

```text
ชื่อเรื่อง:
แหล่งที่มา:
ใจความสำคัญ:
ประโยคหรือแนวคิดที่ชอบ:
สิ่งที่เชื่อมโยงกับชีวิตของฉัน:
คำถามที่อยากศึกษาต่อ:
```

## ตัวอย่างบันทึกสั้น

### ชื่อเรื่อง

การฟังอย่างตั้งใจ

### ใจความสำคัญ

การฟังอย่างตั้งใจคือการฟังเพื่อเข้าใจ ไม่ใช่ฟังเพื่อเตรียมตอบ การฟังที่ดีช่วยให้อีกฝ่ายรู้สึกว่าความรู้สึกของเขามีคุณค่า

### สิ่งที่เชื่อมโยงกับชีวิตของฉัน

บางครั้งเวลาที่เพื่อนเล่าปัญหา ฉันเผลอรีบแนะนำวิธีแก้ ทั้งที่เพื่อนอาจต้องการแค่คนรับฟัง การอ่านเรื่องนี้ทำให้ฉันอยากฝึกถามเพื่อนก่อนว่า "อยากให้ช่วยคิดทางออก หรืออยากให้ฟังก่อน"

### คำถามที่อยากศึกษาต่อ

ถ้าเราเป็นผู้ฟังที่ดี แต่เรื่องของเพื่อนหนักเกินไป เราควรส่งต่อความช่วยเหลืออย่างไรให้เหมาะสม

## สิ่งที่ได้เรียนรู้จากโครงงานนี้

การทำ Reading Journal ทำให้ฉันเห็นว่าความสนใจของตัวเองไม่ได้เกิดจากความรู้สึกชั่วคราว แต่เป็นสิ่งที่ฉันค่อย ๆ ศึกษาและเชื่อมโยงกับชีวิตจริงได้ นอกจากนี้ยังช่วยให้ฉันเขียนอธิบายตัวเองใน Portfolio ได้ชัดเจนขึ้น

## หลักฐานประกอบ

- รายชื่อหนังสือหรือบทความที่อ่าน
- ตัวอย่าง Reflection จากบันทึกบางครั้ง
