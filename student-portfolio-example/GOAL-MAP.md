# Goal Map

ไฟล์นี้ใช้สรุปเส้นทางความสนใจ เป้าหมาย และสิ่งที่ต้องพัฒนาก่อนยื่น Portfolio

## เป้าหมายระยะสั้น

ช่วง ม.5 ถึง ม.6 ฉันต้องการพัฒนาตนเองใน 4 ด้านหลัก

| ด้าน | เป้าหมาย | หลักฐานที่อยากมี |
|---|---|---|
| การเรียน | รักษาผลการเรียนให้สม่ำเสมอ โดยเฉพาะภาษาไทย ภาษาอังกฤษ สังคม และวิทยาศาสตร์ | ใบแสดงผลการเรียน / สรุปวิชาที่เกี่ยวข้อง |
| กิจกรรม | เข้าร่วมกิจกรรมที่เกี่ยวกับการสื่อสาร งานอาสา หรือการดูแลเพื่อน | บันทึกกิจกรรม / ภาพถ่าย / หนังสือรับรอง |
| ผลงาน | ทำผลงานที่สะท้อนความสนใจด้านจิตวิทยาและสื่อสร้างสรรค์ | โครงงาน / คลิป / บทความ / สื่อรณรงค์ |
| การสะท้อนคิด | ฝึกเขียน Reflection หลังทำกิจกรรม | บันทึก Reflection ใน repository นี้ |

## เป้าหมายระยะยาว

ฉันอยากศึกษาต่อในสาขาที่ช่วยให้เข้าใจมนุษย์และสื่อสารกับผู้คนได้ดีขึ้น เพื่อในอนาคตจะสามารถทำงานที่เกี่ยวข้องกับ

- การให้คำปรึกษาเบื้องต้นหรือการดูแลสุขภาพใจ
- การสื่อสารเพื่อสังคม
- การออกแบบกิจกรรมเพื่อพัฒนาผู้เรียน
- การผลิตสื่อที่ช่วยให้คนเข้าใจเรื่องยากได้ง่ายขึ้น

## ความเชื่อมโยงระหว่างความสนใจกับคณะ

| ความสนใจ | คณะที่เกี่ยวข้อง | เหตุผล |
|---|---|---|
| ชอบฟังและเข้าใจความรู้สึกของคน | จิตวิทยา | ต้องการเรียนรู้พฤติกรรม ความคิด และอารมณ์ของมนุษย์อย่างเป็นระบบ |
| ชอบทำสื่อให้เข้าใจง่าย | นิเทศศาสตร์ | สนใจการเล่าเรื่อง การผลิตสื่อ และการสื่อสารกับสังคม |
| ชอบช่วยเพื่อนและทำกิจกรรม | ครุศาสตร์ / ศึกษาศาสตร์ | สนใจการพัฒนาผู้เรียน การออกแบบกิจกรรม และการสื่อสารในห้องเรียน |

## Skill Map

| Communication | ทำสื่อรณรงค์เรื่องสุขภาพใจในโรงเรียน | ฝึกพูดหน้าชั้นและเขียนบทให้กระชับขึ้น |

## คำถามทบทวนตัวเอง

- กิจกรรมใดสะท้อนตัวตนของฉันมากที่สุด
- ผลงานใดแสดงความพยายามและพัฒนาการชัดเจนที่สุด
- สิ่งที่ฉันสนใจเกี่ยวข้องกับคณะที่อยากเรียนอย่างไร
- ถ้ากรรมการอ่าน Portfolio ของฉัน เขาควรเห็นจุดเด่นอะไร
